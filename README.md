# ESP32_Smart_Watch

## Basic implementation of Smart-Watch device on ESP32 chipset with built in display. 

### Used lib's:

1. TinyGPSPlus (Date-Time-Location Sensor)
2. TFT_eSPI (Built-in Display)
3. MAX30105 (IR Sensor)
4. SimpleTimer
5. SoftwareSerial 
6. Wire

### Used Hardware's:

1. LilyGO TTGo T-Display ESP32
2. ATGM336H
3. MAX30105
