# ESP32_Smart_Watch

Basic implementation of Smart-Watch device on ESP32 chipset with built in display. 

## Used lib's:

### TinyGPSPlus (Date / Time / Location operating)
### TFT_eSPI (Built-in Display)
### MAX30105 (IR sensor library)
### SimpleTimer
### SoftwareSerial 
### Wire

## Used Hardware's:

### TinyGPSPlus (Date / Time / Location operating)
### TFT_eSPI (Built-in Display)
### MAX30105 (IR sensor library)
### SimpleTimer
### SoftwareSerial 
### Wire
