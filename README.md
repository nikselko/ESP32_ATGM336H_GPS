# ESP32_Smart_Watch

## C++ implementation of Smart-Watch with GPS and Pulseoximeter. 

### Used lib's:

1. TinyGPSPlus (Date-Time-Location Sensor)
2. TFT_eSPI (Built-in Display)
3. MAX30105 (IR Sensor)
4. SPI
5. SimpleTimer
6. SoftwareSerial 
7. Wire

### Used Hardware's:

1. LilyGO TTGo T-Display ESP32
2. ATGM336H
3. MAX30105
